# nixcfg-derive

derive macro for [nixcfg](https://crates.io/crates/nixcfg). use the
re-exported `#[nixcfg]` attribute from the `nixcfg` crate rather than
depending on this crate directly.

see the [nixcfg-rs README](https://github.com/mushrowan/nixcfg-rs) for
usage examples.
